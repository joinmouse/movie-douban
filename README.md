## 运用技术
- 基本：HTML5 、CSS3、JavaScript
- 库或框架：jQuery、require（组件化代码块）
- webpack、npm（js的打包）

## 实现功能
- 展示电影排行榜Top250
- 展示北美最新电影票房榜
- 搜索电影
